## 概述

columns 是一个使用 Vue3 作为主要技术栈开发的专栏应用，主要使用 Vue3 新增的 Composition API

在线演示地址：[columns](https://columns.mneumi.cn)

对应后端服务器代码：[columns-server](https://github.com/mneumi/columns-server)



## 技术栈

| 技术栈      | 说明         |
| ----------- | ------------ |
| 语言        | TypeScript   |
| 基础框架    | Vue.js 3.x   |
| 路由库      | vue-router@4 |
| 状态管理    | vuex@4       |
| CSS预处理器 | SCSS         |
